# UTube Clone Coding

1. URL Design

Global Routers

> / : Home Page
> /join : Join User Page
> /login : Login Page
> /search : Search Video Page

Users

> /users/:id : Edit Profile
> /users/logout : Log out
> /users/edit : Edit My Profile
> /users/delete : Delete Profile

Videos

> /videos/:id
> /videos/:id/edit
> /videos/:id/delete
> /vidoes/upload

Comments

> /videos/comments
> /videos/commets/delete
